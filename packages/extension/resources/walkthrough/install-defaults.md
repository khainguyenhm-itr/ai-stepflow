## Install default agents & skills

This bundles a ready-made set of SDLC **agents** and **skills** into your global
`~/.claude` directory, plus the project's engineering rules — so you have building
blocks to assemble flows from on day one.

Run **AI StepFlow: Install Default Agents & Skills** from the button on the left
(or the Command Palette).

You can always add your own later: agents and skills are just markdown files under
`~/.claude` (global) or `.claude` (per project).
